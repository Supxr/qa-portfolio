# Test Case: Adding Products to the Shopping Cart

**Test Case ID:** TC_SD_11<br>
**Test Objectives:** To verify that products can be added to the shopping cart.<br>
**Pre-conditions:** The user must be logged into the SauceDemo Website.<br><br>

**Test Steps:**
1. Navigate to https://www.saucedemo.com/
2. Enter username: `standard_user`
3. Enter password: `secret_sauce`
4. Click `Login`
5. Find item: `Swag Labs Backpack`
6. Click `Add to cart`

**Expected Result(s):** The `Add to cart` button changes to `Remove` and the shopping cart badge is updated with a value appropriate to the number of items added.<br>

**Actual Result(s):** The `Add to cart` button changes to `Remove` and the shopping cart badge is updated with a value appropriate to the number of items added.<br>

**Status:** Pass
