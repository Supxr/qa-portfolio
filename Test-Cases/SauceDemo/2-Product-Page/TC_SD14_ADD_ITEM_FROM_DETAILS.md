# Test Case: Adding Product to Shopping Cart from the Product Details Page

**Test Case ID:** TC_SD_14<br>
**Test Objectives:** To verify that the user is able to add a product to the shopping cart from the product details page.<br>
**Pre-conditions:** The user must be logged into the SauceDemo Website and on a product details page.<br><br>

**Test Steps:**
1. Navigate to https://www.saucedemo.com/
2. Enter username: `standard_user`
3. Enter password: `secret_sauce`
4. Click `Login`
5. Find `Sauce Labs Backpack`
6. Click `Sauce Labs Backpack`
7. Click `Add to cart`

**Expected Result(s):** The `Add to cart` button changes to `Remove` and the shopping cart badge is updated to the appropriate number of items.<br>

**Actual Result(s):** The `Add to cart` button changes to `Remove` and the shopping cart badge is updated to the appropriate number of items.<br><br>

**Status:** Pass
