# Test Case: Returning to the Products Page from the Product Details Page

**Test Case ID:** TC_SD_16<br>
**Test Objectives:** To verify that the user is able to return to the Products page from the Product Details page using the `Back to Products` button
**Pre-conditions:** The user must be logged into the SauceDemo website and in a product details page.<br><br>

**Test Steps:** 
1. Navigate to https://www.saucedemo.com/
2. Enter username: `standard_user`
3. Enter password: `secret_sauce`
4. Click `Login`
5. Find `Sauce Labs Backpack`
6. Click `Sauce Labs Backpack`
7. Click `Back to products`

**Expected Result(s):** The user is directed back to the main products page.<br>

**Actual Result(s):** The user is directed back to the main products page.<br><br>

**Status:** Pass
