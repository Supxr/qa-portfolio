# Test Case: Cancelling the Order from the Information Request Form

**Test Case ID:** TC_SD_28<br>
**Test Objectives:** To verify that the user is able to cancel the order from the information request page.<br>
**Pre-conditions:** The user must be logged into the SauceDemo website, and on the information request page. <br>

**Test Steps:**
1. Navigate to https://www.saucedemo.com/
2. Enter username: `standard_user`
3. Enter password: `secret_sauce`
4. Click `Login` button
5. Find Item: `Sauce Labs Backpack`
6. Click `Add to cart` button
7. Click shopping cart icon on the top right
8. Click `Checkout` button
9. Click `Cancel` button

**Expected Result(s):** The user is redirected back to the shopping cart page.<br>

**Actual Result(s):** The user is redirected back to the shopping cart page. <br><br>

**Status:** Pass
