 # Test Case: Redirecting to external website using About button

 **Test Case ID:** TC_SD_18<br>
 **Test Objectives:** To verify that the `About` button redirects the user to https://www.saucelabs.com/<br>
 **Pre-conditions:** The user must be logged into the SauceDemo website.<br><br>

 **Test Steps:** 
 1. Navigate to https://www.saucedemo.com/
 2. Enter username: `standard_user`
 3. Enter password: `secret_sauce`
 4. Click `Login` button
 5. Find hamburger menu on the top left
 6. Click `About`

**Expected Result(s):** The user is redirected to https://www.saucelabs.com/<br>

**Actual Result(s):** The user is redirected to https://www.saucelabs.com/<br><br>

**Status:** Pass
