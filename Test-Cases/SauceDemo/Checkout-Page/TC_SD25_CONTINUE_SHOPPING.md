# Test Case: Returning to the main products page from the checkout page.

**Test Case ID:** TC_SD_25<br>
**Test Objectives:** To verify that the user is able to click the `Continue Shopping` button in the checkout page in order to return to the main products page.<br>
**Pre-conditions:** The user must be logged into the SauceDemo website checkout page.<br>

**Test Steps:**
1. Navigate to https://www.saucedemo.com/
2. Enter username: `standard_user`
3. Enter password: `secret_sauce`
4. Click `Login` button
5. Click shopping cart icon on the top right
6. Click `Continue Shopping` button

**Expected Result(s):** The user is able to return to the main products page from the checkout page by clicking the `Continue Shopping` button.<br>

**Actual Result(s):** The user is able to return to the main products page from the checkout page by clicking the `Continue Shopping` button.<br><br>

**Status:** Pass
