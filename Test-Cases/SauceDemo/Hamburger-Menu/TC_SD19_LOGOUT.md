# Test Case: Logging Out Using the Logout Button in the Hamburger Menu

**Test Case ID:** TC_SD_19<br>
**Test Objectives:** To verify that the user is redirected to the logged out, and redirected to the login page when clicking the logout button.<br>
**Pre-conditions:** The user must be logged into the SauceDemo website.<br><br>

**Test Steps:** 
1. Navigate to https://www.saucedemo.com/
2. Enter username: `standard_user`
3. Enter password: `secret_sauce`
4. Click `Login` button
5. Find hamburger menu on the top left
6. Click `Logout` button

**Expected Result(s):** The user is logged out and redirected to the login page.<br>

**Actual Result(s):** The user is logged out and redirected to the login page.<br><br>

**Status:** Pass
