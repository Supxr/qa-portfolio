# Test Case: Sorting product by using Low-to-High filter

**Test Case ID:** TC_SD_08<br>
**Test Objectives:** To verify that applying the low-to-high sorting filter arranges the products in the correct order.<br>
**Pre-conditions:** The user must be in the products page of the SauceDemo website.<br><br>

**Test Steps:**
1. Navigate to https://www.saucedemo.com/
2. Enter username: `standard_user`
3. Enter password: `secret_sauce`
4. Click `Login` button
5. Click filter dropdown menu on the top right
6. Click `Price (low to high)`

**Expected Result(s):** The system arranges the products according to ascending price.<br><br>

**Actual Result(s):** The system arranges the products according to ascending price.<br><br>

**Status:** Pass
