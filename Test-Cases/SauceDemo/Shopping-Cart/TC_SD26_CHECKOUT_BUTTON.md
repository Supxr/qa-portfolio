# Test Case: Accessing the Checkout Page from the Shopping Cart Page

**Test Case ID:** TC_SD_26<br>
**Test Objectives:** To verify that the user can proceed to the checkout page from the shopping cart page with at least one item.<br>
**Pre-conditions:** The user must be logged into the SauceDemo website, on the shopping cart page, with at least one item in the cart.<br><br>

**Test Steps:**
1. Navigate to https://www.saucedemo.com/
2. Enter username: `standard_user`
3. Enter password: `secret_sauce`
4. Click `Login` button
5. Find `Sauce Labs Backpack`
6. Click `Add to cart` button
7. Click shopping cart icon on the top right
8. Click `Checkout` button

**Expected Result(s):** The user is transferred to the checkout information request form.<br>

**Actual Result(s):** The user is transferred to the checkout information request form.<br><br>

**Status:** Pass
